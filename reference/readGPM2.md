# readGPM2

read peatland area from GPM2

## Usage

``` r
readGPM2(subtype = "1km")
```

## Arguments

- subtype:

  resolution ("1km" or "500m")

## Value

List of magpie objects with results on cellular level, weight, unit and
description.
