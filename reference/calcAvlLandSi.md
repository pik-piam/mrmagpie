# calcAvlLandSi

Extracts si0 and nsi0 areas based on Ramankutty dataset

## Usage

``` r
calcAvlLandSi(cells = "lpjcell")
```

## Arguments

- cells:

  magpiecell (59199 cells) or lpjcell (67420 cells)

## Value

magpie object in cellular resolution
