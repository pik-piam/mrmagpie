# readLabourProdImpactEmu

read in labour productivity impacts from climate change emulated by the
LAMACLIMA project

based on method of Orlov et al. 2019. Economics of Disasters and Climate
Change, 3(3), 191-211.

## Usage

``` r
readLabourProdImpactEmu()
```

## Value

magpie object of gridded productivity loss in percent (0-100)
