# calcLuh2SideLayers

Function extracts biodiversity data for LUH2 land cover types

## Usage

``` r
calcLuh2SideLayers(cells = "lpjcell")
```

## Arguments

- cells:

  number of cells to be returned: magpiecell (59199), lpjcell (67420)

## Value

magpie object in cellular resolution
