# readGridPopGao

Read gridded population, by urban and rural, from Gao O'Neill and JOnes
dataset, see
https://www.cgd.ucar.edu/iam/modeling/spatial-population-scenarios.html
https://doi.org/10.7927/m30p-j498

## Usage

``` r
readGridPopGao(subtype = "future")
```

## Arguments

- subtype:

  only "future" post-2000 available for this source
