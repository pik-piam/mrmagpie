# calcRRLayer

Function extracts range-rarity as used for biodiversity loss

## Usage

``` r
calcRRLayer(cells = "lpjcell")
```

## Arguments

- cells:

  number of cells to be returned: magpiecell (59199), lpjcell (67420)

## Value

magpie object in cellular resolution
