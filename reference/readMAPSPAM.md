# readMAPSPAM

Reads the MAP-SPAM crop data per year (mapping each year different)

## Usage

``` r
readMAPSPAM(subtype = "harvested")
```

## Arguments

- subtype:

  It can be either "harvested" or "physical" area

## Value

magpie object with croparea data in ha
