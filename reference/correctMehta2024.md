# correctMehta2024

correct Global Area Equipped for Irrigation Dataset 1900-2015 from Mehta
et al., 2024

## Usage

``` r
correctMehta2024(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

magpie object in cellular resolution
