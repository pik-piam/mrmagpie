# correctLeifeld2018

correct potential peatland area from Leifeld2018

## Usage

``` r
correctLeifeld2018(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

List of magpie objects with results on cellular level, weight, unit and
description.
