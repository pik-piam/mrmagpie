# calcNpiNdcAdAolcPol

Function creates dummy NPI/NDC policies

## Usage

``` r
calcNpiNdcAdAolcPol(cells = "lpjcell")
```

## Arguments

- cells:

  lpjcell for 67420 cells or magpiecell for 59199 cells

## Value

magpie object in cellular resolution
